# Data Science Projects with Python - Stephen Klosterman

This directory contains projects based on the book:

Data Science Projects with Python: A case study approach to gaining valuable insights from real data with machine learning - Stephen Klosterman

The focus of this book is applying Data Science techniques to real-world datasets and building complete Machine Learning workflows.

---

## 📂 Directory Structure

chapters  
Study materials and chapter implementations.

projects  
Complete Data Science projects.

datasets  
Datasets used for modeling and experiments.

---

## 📊 Topics Covered

- Data exploration
- Data cleaning
- Feature engineering
- Model evaluation
- Logistic regression
- Decision trees
- Random forest
- Gradient boosting
- XGBoost
- Model interpretation
