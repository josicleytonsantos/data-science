# data-science-journey

This repository documents my practical journey in **Data Science**, combining structured study, hands-on implementations, and applied projects.

The repository is organized around well-known books in the field and real-world datasets, aiming to build a consistent and transparent learning portfolio.

The focus is to develop skills in:

- Data analysis
- Statistical reasoning
- Machine learning
- Data manipulation
- Practical problem solving

---

## 🎯 Objectives

- Study core Data Science concepts through recognized books
- Implement examples and exercises
- Apply techniques to real datasets
- Develop practical projects
- Build a public Data Science portfolio

---

## 🛠 Tools and Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Repository Structure

Each book follows the same internal organization:

chapters  
Contains exercises, notes, and implementations from each chapter.

projects  
Contains practical applications based on the concepts studied.

datasets  
Contains datasets used in experiments and analysis.

Books included in this repository:

| Book | Focus |
|------|------|
| Python for Data Analysis | Data manipulation and analysis |
| Data Science from Scratch | Mathematical foundations and algorithms |
| Data Science Projects with Python | Applied machine learning projects |

Additionally, the repository contains a dedicated section for independent work:

personal-projects

This section includes projects developed outside the books using real-world datasets.

---

## 📊 Study Methodology

For each book, the following workflow is applied:

1. Study the chapter concepts
2. Reproduce examples
3. Implement exercises
4. Test with different datasets
5. Develop small applied projects
