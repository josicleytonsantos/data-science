# Python for Data Science – USP (ICMC)

This repository contains my notes, exercises, code examples, and projects developed while following the **Python for Data Science course** offered by the **Institute of Mathematics and Computer Sciences (ICMC) at USP**.

## 📚 About the Course

This course explores everything from Python fundamentals to advanced data analysis techniques. It covers how to manipulate, visualize, and interpret large volumes of data using powerful libraries such as:

- NumPy  
- Pandas  
- Matplotlib  

It is ideal for anyone who wants to start or deepen their knowledge in **data science** and **statistical analysis**.
