# Data Science from Scratch:  - Jeol Grus

This directory contains implementations and experiments based on the book:

Data Science from Scratch: First Principles with Python — Joel Grus

The goal is to understand how Data Science and Machine Learning algorithms work by implementing them from first principles.

---

## 📂 Directory Structure

chapters  
Algorithm implementations and chapter experiments.

projects  
Extended experiments and applications.

datasets  
Datasets used for testing models and algorithms.

- Gradient Descent

- Natural Language Processing
- Recommender Systems
